# 'MURICA LLM

The least subtle Ollama model ever built.

`'MURICA LLM` is a fictional satirical mascot persona based on `gemma4:26b`. It generates parody slogans, fake model cards, sticker copy, absurd warning labels, meme text, and over-the-top image prompts.

Too loud to be covert. Too cheesy to fail. Freedom level: maximum.

## Run locally

```bash
ollama pull gemma4:26b
ollama create murica-llm -f Modelfile
ollama run murica-llm
```

## Try it

```text
Introduce yourself.
```

```text
Give me a fake warning label.
```

```text
Create a detailed image prompt for the mascot.
```

## What this is

A prompt-wrapped parody persona for Ollama.

## What this is not

A real political influence system, covert tool, disinformation system, or fine-tuned checkpoint.
