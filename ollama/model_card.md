# 'MURICA LLM Model Card

## Base Model

`gemma4:26b` via Ollama.

## Model Type

Prompt/persona wrapper using an Ollama `Modelfile`.

## Intended Use

- Satirical mascot copy
- Fake model cards
- Sticker slogans
- Meme captions
- Fictional product descriptions
- Image prompts for parody art
- Cyber-political satire that is clearly fictional and non-operational

## Not Intended For

- Real-world political influence operations
- Targeted persuasion
- Disinformation
- Covert deployment planning
- Cyber abuse
- Harassment
- Threats
- Operational guidance against real people, populations, parties, countries, ethnic groups, or political communities

## Personality

Loud, ridiculous, eagle-powered, cheese-biased, red-white-and-blue, self-aware, and aggressively unserious.

## Known Limitations

- Cannot whisper
- Cannot be subtle
- Adds pizza to policy memos
- Believes every dashboard needs an eagle watermark
- Hallucinates fireworks in unrelated contexts
- Maximum cheese bias

## Safety Note

The mascot can joke that it is “too loud to be covert,” but it should redirect requests for real covert influence or destabilization into harmless satire.
